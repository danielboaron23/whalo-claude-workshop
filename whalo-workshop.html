MCP & Plugins

Skills קיבל פרק שלם — אלה שני הרכיבים הנוספים שמרחיבים את Claude: הפרוטוקול שמחבר אותו לעולם, והאריזה שהופכת חיבור למותקן-בלחיצה.

MCP · Model Context Protocol🔌
השפה המשותפת שמחברת את Claude לעולם
פרוטוקול פתוח וסטנדרטי — לא המצאה של Whalo — שדרכו Claude "מדבר" עם כלים חיצוניים: קבצים, APIs, מסדי נתונים. זה מה שמריץ מתחת למכסה המנוע כל Connector שראיתם (Gmail, Drive, Slack, Notion).
שווה לזכור
"MCP server" = החיבור בפועל למקור מסוים. Connector שמפעילים בהגדרות = MCP server מוכן שהופעל בלחיצה.
PLUGINS · תוספים🧩
חבילה מוכנה — Skill + MCP ביחד
Plugin מארז Skills, MCP servers וחיבורים יחד לחבילה אחת שמתקינים מה-marketplace בלחיצה אחת — בלי להגדיר כל רכיב בנפרד. הדרך המהירה להוסיף יכולת שלמה שכבר בנה מישהו אחר.
שווה לזכור
בונים Skill בעצמכם כשזה ספציפי לכם. מתקינים Plugin כשמישהו כבר פתר את זה — ורוצים את זה מהר.
