# Store character limits

> זה ה"ידע הכללי" — Claude יודע אותו, אבל מצמידים אותו כאן כדי שהספירה תהיה **מדויקת
> ועקבית** ולא תלויה בזיכרון. כל שדה נספר ומאומת מול הטבלה הזו.

## Apple App Store
| Field | Limit (chars) |
|---|---|
| App name (title) | 30 |
| Subtitle | 30 |
| Promotional text | 170 |
| Description | 4000 |

## Google Play
| Field | Limit (chars) |
|---|---|
| App title | 30 |
| Short description | 80 |
| Full description | 4000 |

## How to count
- Count **visible characters including spaces**.
- Emoji and most Hebrew characters count as 1 each for store purposes.
- If a brief specifies different limits, the **brief overrides** this file.
- Always show `(used/limit)` next to each field so it's obvious nothing overflows.
