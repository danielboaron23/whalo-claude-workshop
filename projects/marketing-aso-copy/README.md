# 📂 וריאציות קופי לחנות — ASO Copy

כלי לדוגמה לצוות **Marketing** · נבנה ב-Claude Code · חלק מסדנת Whalo.

## מה זה עושה
מבריף קצר על משחק → 3-4 גרסאות קופי לחנות (title/subtitle/short), כל אחת בטון אחר,
עם ספירת תווים מאומתת מול מגבלות החנות.

## מבנה
```
marketing-aso-copy/
├── GOAL.md · README.md · STEPS.md
├── data/game-brief.md          ← הבריף (input לדוגמה)
├── app/index.html              ← האפליקציה (פותחים בדפדפן)
└── .claude/skills/aso-copy/SKILL.md
```

## איך מריצים
פותחים את `app/index.html` בדפדפן. רץ offline.

## 🔌 חיבור (Google Drive)
בעבודה אמיתית הבריפים יושבים ב-Google Drive. עם ה-connector אפשר לבקש מ-Claude:
*"קרא את הבריף מה-Drive וצור 4 וריאציות קופי לחנות"* — בלי להעתיק ידנית.

## הדפוס שלוקחים
**בריף + אילוצים (אורך/טון/קהל) → מגוון פלטים מובנה.** עובד לכל כתיבה שיווקית:
מיילים, פוסטים, כותרות, מודעות.
