# 🤝 דמו Cowork — טריאז' פידבק שחקנים

> בניגוד ל-labs האחרים (שבונים Skill ב-Claude Code) — כאן **אין בנייה**.
> זו הדגמת Cowork: תיקייה + בקשה אחת → תוצר מוכן. הכל בתוך אפליקציית Claude.

---

## 🎯 המטרה
לקחת 20 פניות שחקנים אמיתיות-לדוגמה, במגוון פורמטים ומקורות, ולהפוך אותן
תוך דקות ל**טבלת טריאז'** ברורה שאפשר להעביר לצוות המוצר היום.

## 💡 למה זה עוזר
- חוסך שעות של קריאה ומיון ידני בין ערוצים.
- **מזהה כפילויות** — 3 אנשים מדווחים על אותו קראש = תקלה קריטית, לא 3 מקרים נפרדים.
- הופך "ערימת טקסט" ל**החלטה** — מה לתקן קודם.

## 📋 מה מכינים
- התיקייה `data/tickets/` — 20 קבצי טקסט (App Store, Google Play, מיילים, Discord, טופס פידבק).
- שום חיבור/MCP נדרש — הקבצים כבר בתיקייה מקומית.

## ▶️ איך עושים (בתוך אפליקציית Claude — בלי טרמינל)

1. פתחו את אפליקציית **Claude** ← מצב **Cowork**.
2. **גררו את כל תיקיית** `data/tickets/` (או את כל 20 הקבצים) לתוך השיחה.
3. הדביקו את הבקשה:

```
יש לי כאן 20 פניות שחקנים ממשחק מובייל, ממקורות שונים (ביקורות
חנות, מיילים, דיסקורד, טופס פידבק) — בלי שום סדר.

תעבור על כולן ותבנה לי טבלת טריאז':
- קטגוריה (באג / בקשת פיצ'ר / תלונה / שבח)
- חומרה (גבוהה / בינונית / נמוכה) — לבאגים ותלונות בלבד
- כמה פעמים דווחה אותה בעיה בפועל (זהה כפילויות!)
- מקור

בסוף: 3 שורות "Top Priority" — מה הכי דחוף להעביר לצוות המוצר היום, ולמה.
```

4. Claude קורא את כל הקבצים, מזהה שהתקלה של "קריסה בפתיחת התיבה היומית"
   ושל "אובדן יהלומים אחרי רכישה" **כל אחת דווחה פעמיים ממקורות שונים** —
   ומעלה את החומרה שלהן בהתאם.

## ✅ איך בודקים
- כל 20 הפניות מסווגות? שום דבר לא "נפל בין הכיסאות"?
- הכפילויות (קריסת התיבה, אובדן היהלומים) זוהו כתקלה **אחת** עם 2 דיווחים — לא 2 שורות נפרדות?
- ה-Top 3 הגיוניים — משלבים חומרה גבוהה **וגם** תדירות גבוהה, לא רק אחד מהשניים?
- 💡 שדרוג לדיון: *"עכשיו תעשה את זה כל בוקר אוטומטית ותשלח לי ב-Slack"* —
  זה בדיוק המעבר מ-Cowork חד-פעמי ל**תזמון**, בלי לבנות שום כלי.

---

## 🎓 מה זה מדגים בסדנה
- Cowork עובד מצוין על **דאטה מהחיים האמיתיים** — לא רק טבלאות מסודרות.
- "זיהוי כפילויות" הוא בדיוק סוג המשימה שגוזלת שעות ידנית ולוקחת שניות כאן.
- אין כאן שום Skill / MCP / קוד — רק תיקייה ובקשה אחת בשפה חופשית.
